# Prompt Toolkit - Checkpoint 2

Este projeto é um Toolkit de Engenharia de Prompt desenvolvido para a **FIAP**. Ele automatiza a execução de tarefas utilizando diferentes técnicas de prompt (Few-Shot, Role Play, Chain of Thought) e gera relatórios de desempenho e custo.

## 🚀 Como Instalar e Rodar

Siga os passos abaixo para configurar o ambiente e executar o projeto em sua máquina.

### 1. Pré-requisitos
* **Python 3.11+** instalado.
* Uma chave de API da **Groq** (obtenha em [console.groq.com](https://console.groq.com/)).

### 2. Clonar o Repositório
```bash
git clone https://github.com/PedroHNeves1505/checkpoint-2-PromptToolkit-1
cd checkpoint-2-PromptToolkit

### 3. Instalar dependências 
pip install -r requirements.txt

### 4. Configurar Variáveis de Ambiente
### No arquivo .env
GROQ_API_KEY=sua_chave_aqui

### 5. Executar Projeto
python main.py
